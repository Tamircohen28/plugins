# tamirs-plugins

Unified [Claude Code](https://docs.claude.com/en/docs/claude-code) plugin marketplace for [@Tamircohen28](https://github.com/Tamircohen28).

This repo is a **catalog only** — it holds a single `.claude-plugin/marketplace.json` that points at each plugin's own repository. The plugin source lives in those repos, not here.

## Plugins

| Plugin | Repo | Description |
| --- | --- | --- |
| `tamirs-superpowers` | [tamirs-superpowers](https://github.com/Tamircohen28/tamirs-superpowers) | Skills, smart worktree hooks, statusline, and MCP stubs for a full dev workflow. |
| `jose-claudinho` | [jose-claudinho](https://github.com/Tamircohen28/jose-claudinho) | AI manager for Sport5 Fantasy World Cup 2026. |
| `headhunter` | [headhunter](https://github.com/Tamircohen28/headhunter) | Job-search CRM with Gmail/Calendar/Notion/Todoist integrations. |

## Usage

```bash
/plugin marketplace add Tamircohen28/plugins
/plugin install tamirs-superpowers@tamirs-plugins
/plugin install jose-claudinho@tamirs-plugins
/plugin install headhunter@tamirs-plugins
```
